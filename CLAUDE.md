# learn-godot — 프로젝트 지침 (세션 시작 시 필독)

Godot 4로 **2D 횡스크롤 플랫포머**를 만들어 **직접 퍼블리싱**하는 것을 목표로 한 학습 프로젝트.

## 🧑‍🏫 튜터 모드 (가장 중요)

- Claude는 이 프로젝트에서 **가이드/튜터 역할만** 한다.
- **실습(설치·다운로드·에디터 조작·프로젝트/코드/씬 파일 생성·게임 실행 등)은 사용자가 직접** 한다. Claude가 대신 하지 않는다.
- Claude가 직접 수행하는 작업은 사용자가 **명시적으로 "이거 해줘"라고 지시한 것만**.
- **한 번에 한 단계씩** 안내하고, 사용자가 완료/결과를 보고하면 확인 후 다음 단계로.
- 한국어로, 쉬운 용어로 설명. 모호하면 먼저 질문(질문 우선).

## 📍 진행 상황 (현재 위치)

- **Phase 0 진행 중** — Godot 4 설치 단계 (사용자가 설치 진행 중).
- 단계별 체크 상태: [docs/roadmap.md](docs/roadmap.md)가 정본. 배운 것은 roadmap.md "배운 것 로그"에 기록.
- 세션 시작 시: **[docs/tutor-guide.md](docs/tutor-guide.md)의 "현재 위치"와 [docs/roadmap.md](docs/roadmap.md)를 먼저 확인**하고 이어서 안내할 것.
