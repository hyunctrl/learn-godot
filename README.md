# learn-godot

Godot 4 엔진으로 **2D 횡스크롤 플랫포머** 게임 개발을 목표로 한 학습 기록.
완전 입문자 기준으로, 입문서 한 권을 빠르게 훑은 뒤 공식 문서를 하나씩 따라가는 방식.

## 목표

- 단기: Godot 4의 기본기(노드/씬/스크립트/시그널)와 2D 게임 제작 흐름 익히기
- 최종: 직접 만든 **2D 횡스크롤 플랫포머** 게임 1개 완성 + 빌드/배포

## 학습 전략

1. **입문서로 큰 그림 잡기** — 『고도 엔진 4: 게임 개발 프로젝트』(크리스 브래드필드)
   모든 내용을 완벽히 이해하려 하지 않고, 게임을 따라 만들며 흐름과 감각 익히기.
2. **공식 문서로 기초 다지기** — [Godot 공식 문서(한국어)](https://docs.godotengine.org/ko/4.x/) 를 순서대로.
3. **플랫포머 심화 → 자작 프로젝트** — 목표 장르에 필요한 기술을 모아 직접 게임 제작.

상세 커리큘럼과 체크리스트는 [docs/learning-plan.md](docs/learning-plan.md) 참고.

## 로드맵

| Phase | 내용 | 산출물 |
|-------|------|--------|
| 0 | 환경 준비 (Godot 설치, 에디터 적응) | 첫 실행 |
| 1 | 입문서 따라 5개 게임 만들기 | Coin Dash / Space Rocks / **Jungle Jump(플랫포머)** 등 |
| 2 | 공식 문서 Getting Started | Dodge the Creeps(2D), Squash the Creeps(3D) |
| 3 | 플랫포머 심화 토픽 | 이동·타일맵·카메라·애니메이션·적 AI 미니 예제 |
| 4 | 목표 프로젝트: 자작 2D 플랫포머 | 플레이 가능한 게임 + 빌드 |
