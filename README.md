# learn-godot

Godot 4 엔진으로 **2D 횡스크롤 플랫포머**를 개발하고 **직접 퍼블리싱**하기까지의 학습 기록.

## 🎯 목표 (북극성)

> **대학 졸업 스타트업 수준으로 게임을 개발해, 직접 출시(퍼블리싱)하는 경험을 갖는다.**

완벽한 이해보다 *완성과 출시*가 우선. 모든 학습 단계는 "출시 가능한 게임 한 편"으로 수렴한다.

## 학습 전략

1. **입문서로 큰 그림** — 『고도 엔진 4: 게임 개발 프로젝트』(크리스 브래드필드). 따라 만들며 흐름·감각 익히기(이해 80%면 OK).
2. **공식 문서로 기초** — [Godot 공식 문서(한국어)](https://docs.godotengine.org/ko/4.x/) 순서대로.
3. **심화 → 자작 → 출시** — 플랫포머 기술을 모아 직접 게임을 만들고 itch.io/Steam에 퍼블리싱.
4. **가지치기 보강** — 메인 중 모르는 인접 분야(기획·디자인·레벨·QA·이론)가 나오면 그때그때 보강.

| Phase | 내용 | 산출물 |
|-------|------|--------|
